# Command palette

Open the palette with `p` or `:`.

It provides a fast path for common commands without leaving the keyboard flow.

## Current commands

- `refresh`
- `reload`
- `help`
- `copy`
- `inspect`
- `renew`
- `quit`

## Behavior

The palette stays intentionally small in v1.

It is meant to reduce friction, not to become a second user interface inside the TUI.
