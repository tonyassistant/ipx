# Keybindings

## Navigation

- `k` / `â†‘` move up
- `[` / `]` switch inspector views

## Palette

- `p` or `:` open the palette
- `Esc` close the palette
- `Enter` run the typed command

## Actions

- `a` move up in the action list
- `s` move down in the action list
- `Enter` run or confirm
- `Esc` cancel confirmation

## Session

- `r` refresh
- `q` quit
