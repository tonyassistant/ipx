# ipx

ipx is a keyboard-first macOS network operations console built for engineers.

It is designed to feel sharp, fast, and trustworthy, with a terminal experience closer to modern operator tools than to a pile of disconnected shell commands.

## v1 scope

Version 1 focuses on a clean inspect-first workflow:

- interface inventory
- inspector-driven details
- event log visibility
- command palette interaction
- polished keyboard-first navigation

## Design intent

ipx is being built with a few hard constraints:

- clarity beats cleverness
- reads come before writes
- risky actions must be explicit
- the interface should feel calm under pressure
- the tool should be useful even before mutation features expand

## Why Rust

Rust is the right fit for this tool because it gives us:

- strong terminal app ergonomics
- a credible path to a serious local binary
- reliable systems-facing behavior
- good testing and release discipline
