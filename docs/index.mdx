# ipx

ipx is a keyboard-first network operations tool for macOS.

It is designed for engineers who want a fast, inspectable, low-friction interface for understanding and eventually controlling network state without digging through layered system menus.

## Why ipx exists

Modern network tooling on macOS is fragmented.

Important state is spread across:
- System Settings
- `ifconfig`
- `networksetup`
- Wi-Fi utilities
- route and DNS tooling

ipx aims to pull those surfaces into one focused operator experience.

## Product direction

The long-term goal is a documentation-backed operational console for:
- interface inspection
- service visibility
- Wi-Fi and Ethernet workflows
- DHCP and DNS troubleshooting
- safe network mutations with explicit confirmation
