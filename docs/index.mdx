# ipx

ipx helps you inspect macOS network interfaces from a clean keyboard-first terminal interface.

It is built for people who want quick visibility into interface state without jumping through multiple system panels and shell commands.

## What ipx is for

Use ipx when you want to:
- see your network interfaces in one place
- inspect connection state quickly
- understand which interface is active
- stay in the terminal while troubleshooting

## What makes it different

ipx is designed to feel:
- focused
- fast
- readable
- calm under pressure

The goal is not to expose every system detail at once.
The goal is to make the important information easy to read and easy to navigate.
