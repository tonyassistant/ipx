# ipx

ipx is a keyboard-first network console for macOS.

It gives you one clean place to inspect interface state, move quickly, and stay oriented while troubleshooting.

## Why use it

ipx is for moments when you want less hunting and more clarity.

Instead of bouncing between settings panels and shell commands, you get a focused terminal interface built for quick inspection.

## What you can do

- scan available interfaces
- inspect connection state
- move through details quickly
- use a command palette for common actions
- keep context through a visible event log

## Design

ipx is built to feel:
- sharp
- quiet
- readable
- dependable

The interface should help you think clearly, especially when the network is not behaving.
