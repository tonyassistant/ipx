# Installation

## Install with Homebrew

ipx is distributed through the `tonyassistant/homebrew-tap` tap.

```bash
brew tap tonyassistant/homebrew-tap
brew install ipx
```

If you prefer a single command:

```bash
brew install tonyassistant/homebrew-tap/ipx
```

## Requirements for local development

You need:

- Rust toolchain
- macOS, Linux, or Windows for platform-native discovery

macOS currently has the richest discovery model in v1.
Linux and Windows already expose useful interface, address, gateway, and virtual-interface data, with deeper parity still in progress.

## Run ipx from source

```bash
cargo run
```

## Build a release binary

```bash
cargo build --release
```

## Validate your checkout

```bash
cargo fmt
cargo test
cargo build
```
