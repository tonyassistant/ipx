# Quickstart

## Local docs preview

From the repository root:

```bash
cd docs
mint dev
```

## What you should expect

The Mintlify preview gives you:
- live reload
- navigation validation
- early feedback on docs structure
- a fast environment for iterating on content

## Current repository shape

Right now, the docs are the main asset being rebuilt.

The product itself is in a fresh-start state, so the documentation is being used to define:
- product intent
- information architecture
- future implementation direction
