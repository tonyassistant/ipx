# Quickstart

## Run the docs locally

```bash
cd docs
mint dev
```

## Run the current app locally

```bash
cargo run
```

## Run quality checks

```bash
cargo fmt
cargo test
cargo build
```

## Current keyboard controls

- `j` / `↓`: move down
- `k` / `↑`: move up
- `p` or `:`: open command palette
- `r`: request refresh
- `[` / `]` or tab cycle detail views
- `q`: quit
