# Quickstart

## Launch

```bash
cargo run
```

## Core controls

- `j` / `↓` move down
- `k` / `↑` move up
- `[` / `]` switch inspector views
- `p` or `:` open the command palette
- `r` refresh
- `q` quit

## Interface layout

ipx is built around four stable regions:

- interface list
- inspector
- event log
- status bar

That layout keeps the important state visible while you move.
