# Quickstart

## Start ipx

```bash
cargo run
```

## Move around

- `j` / `↓`: move down
- `k` / `↑`: move up
- `[` / `]`: switch inspector views
- `p` or `:`: open the command palette
- `r`: refresh
- `q`: quit

## What you will see

The interface is split into a few core regions:
- a list of interfaces
- an inspector panel
- an event log
- a status footer

This makes it easy to move through interfaces and keep context while you inspect details.
