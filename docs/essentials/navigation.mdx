# Interface guide

## Interface list

The list is your starting point.

Move through interfaces and keep the current selection visible at all times.

## Inspector

The inspector shows details for the selected interface.

Current views:
- overview
- signals
- actions

## Event log

The event log records important feedback so you can see what the app has done and what it is responding to.

## Command palette

Open the palette with `p` or `:`.

Current commands:
- `refresh`
- `reload`
- `help`
- `copy`
- `inspect`
- `quit`
