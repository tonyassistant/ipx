# Using the interface

## Interface list

The left side shows your known network interfaces.

This is where you move first when you want to inspect another interface.

## Inspector

The main inspector shows details for the currently selected interface.

Current views include:
- overview
- signals
- actions

## Event log

The event log records important interaction feedback so you can see what the app is doing.

## Command palette

Open the command palette with `p` or `:`.

Current commands include:
- `refresh`
- `reload`
- `help`
- `copy`
- `inspect`
- `quit`
