# Product vision

ipx should feel like a serious operator console for macOS networking.

Not flashy. Not noisy. Not toy-like.

## Experience goals

- immediate orientation
- fast keyboard movement
- useful defaults
- visible system state
- event logging that builds confidence

## Vibe

The target feel is closer to Codex/OpenCode style terminal products:

- dense but readable
- modern without becoming decorative
- operationally calm
- highly keyboardable
